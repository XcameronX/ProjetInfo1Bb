# Dossier Media

Ce dossier doit contenir les fichiers multimédia suivants :

- **photo.jpg** : Une photo personnelle (format JPG)
- **audio.mp3** : Un fichier audio au format MP3
- **video.mp4** : Une vidéo au format MP4

## Instructions

1. Ajoutez vos propres fichiers multimédia dans ce dossier
2. Renommez-les avec les noms exacts ci-dessus
3. Les fichiers seront automatiquement référencés par les pages HTML du site

**Note** : Ce fichier README.md peut être supprimé après avoir ajouté vos fichiers média.
